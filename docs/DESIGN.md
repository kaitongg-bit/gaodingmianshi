# 稿定面试 · InterviewScript — 设计规范（Notion 风）

本文档与 [PRD.md](../PRD.md) 配合使用；实现时代码侧见 `web/src/app/globals.css` 中的 CSS 变量。

---

## 1. 设计原则

- **极简**：浅底、细线、少装饰；无重渐变。  
- **可读**：字号层级少，行高舒适；中文与英文同一套 scale。  
- **像 Notion**：块感、轻阴影、工具感而非营销轰炸。  
- **一致性**：圆角、边框、间距全部来自 tokens，避免魔法数。

---

## 2. Design Tokens

| Token | 值（建议） | 用途 |
|-------|------------|------|
| `--background` | `#f7f6f3` | 页面底（Notion 偏暖灰） |
| `--surface` | `#ffffff` | 卡片、栏背景 |
| `--surface-muted` | `#f1f0ed` | 悬停、次要区 |
| `--border` | `#e6e4e0` | 分割线、边框 |
| `--border-strong` | `#d4d2cd` | 聚焦、选中 |
| `--text` | `#37352f` | 主文案（Notion 棕灰） |
| `--text-muted` | `#787774` | 辅助说明 |
| `--accent` | `#2383e2` | 主按钮、链接（可替换为品牌色） |
| `--accent-muted` | rgba(35,131,226,0.12) | 选中背景 |
| `--radius-sm` | `4px` | 输入框、小按钮 |
| `--radius-md` | `6px` | 卡片、面板 |
| `--shadow-sm` | `0 1px 2px rgba(15,15,15,0.06)` | 浮层、下拉 |
| `--font-sans` | `ui-sans-serif, system-ui, "Inter", "PingFang SC", "Microsoft YaHei", sans-serif` | 西文优先 + 中文回退 |
| `--space-1` ~ `--space-6` | `4,8,12,16,24,32px` | 间距刻度 |

**排版**

- 页面标题：`20px` / semibold  
- 区块标题：`14px` / semibold  
- 正文：`14px` / regular，`line-height` 1.5–1.6  
- 辅助：`12px` / muted  

**图标**：线性、单色，stroke 约 `1.5px`。

## 3. 信息架构（网页优先）

### 3.1 官网首页（Landing）

**Above the fold**

- 左（或中）：产品名 + 一句价值 + 副文案（合规准备，非考场作弊）。  
- 右（或下）：三栏界面示意（静态图或短视频位）。  
- 主 CTA：`tutorial.cta.demo` — 进入模拟体验。  
- 次 CTA：登录 / 注册。

**首屏下方（与 PRD §3.1.3 一致）**

- **定价 / 购买会员**：卡片或表格展示套餐，Notion 风白底细边框；主按钮跳转结账流程。  
- **分享得次数**：短规则说明 + 次要按钮「参与」；与应用顶栏入口文案统一。

**页脚**：隐私、条款、语言切换。

### 3.2 第 0 页（设置 + 前析）

与 [PRD.md §3.1.1](../PRD.md) 一致：**桌面**建议**左右分栏**——左为简历**全文**滚动区（min 宽约 36%～44%），右为 JD、匹配度、AI 分析、轮数与问题比例、附加要求及主 CTA；**加载中**在右侧或底部通栏展示分步/流式「AI 思考披露」短文案（Notion 风浅底卡片 + 细边框）。**窄屏**改为上下堆叠或 Tab「简历 / 分析与设置」。

### 3.3 应用壳（App Shell · 顶栏 + 轮次 Tab + 三栏）

**最顶栏**（详见 [PRD.md §3.1.2](../PRD.md)）：左 Logo/回首页；**中**为可横向滚动的 **第0页 | 一面 | 二面 | …**（轮数由第 0 页配置决定）；右为 **语言**、**购买会员**、**分享得次数**、**用户菜单**（含「我的逐字稿」）。

**桌面（≥1024px）三栏**（仅适用于**一面～N 面** Tab；**第 0 页** Tab 为双栏前析布局，见 §3.2）

```
┌────────────────────────────────────────────────────────────────────────────┐
│ [Logo]     第0页 | 一面 | 二面 | 三面          语言 会员 分享 [头像▼]      │
├──────────────┬─────────────────────────────┬───────────────────────────────┤
│ [截图解析][+] │  AI 对话（当前选中题）        │  逐字稿（全局，可编辑）        │
│ ──────────── │                             │  [导出 Markdown]              │
│  问题列表     │                             │                               │
│  （滚动）     │                             │                               │
└──────────────┴─────────────────────────────┴───────────────────────────────┘
```

- **左栏**：**顶部固定工具区** — 截图解析、新建问题；**下方**可滚动问题列表。点击列表项 → 中栏切换该题对话 → 右栏滚动并高亮对应段落。  
- **左栏宽度**：`min 240px`, `max 320px`，可折叠为图标条（可选 V1）。  
- **中栏**：flex grow，对话区底部输入框固定。  
- **右栏**：`min 280px`, `max 400px`，内容为**跨轮汇总**的全局稿（题号标明轮次可选）。

**平板 / 手机（<1024px）**

- 顶部 Tab：**问题 | 对话 | 逐字稿** 切换同一数据源。  
- 问题列表可为左侧抽屉（从边滑出）；抽屉内顺序仍为**上工具、下列表**。

### 3.4 「我的逐字稿」（项目库）

卡片/列表展示每个 JD 项目（公司+岗位、更新时间、进度）；点击进入该项目的完整 Tab 与三栏状态。新建项目入口放在列表页顶或空状态内。见 [PRD.md §3.1.3](../PRD.md)。

## 4. 新手教程（分步稿 + i18n key）

教程采用 **遮罩 + 高亮目标 + 底部/侧文案 + 上一步/下一步/跳过**。

| 步骤 | 高亮区域 | i18n key（文案在 `messages/*.json`） |
|------|----------|-------------------------------------|
| 1 | 第 0 页「生成问题」区块 | `onboarding.step1` |
| 2 | 顶部 Tab「一面」 | `onboarding.step2` |
| 3 | 左侧问题列表首题 | `onboarding.step3` |
| 4 | 中间对话输入区 | `onboarding.step4` |
| 5 | 右侧逐字稿编辑区 | `onboarding.step5` |
| 6 | 导出 Markdown 按钮 | `onboarding.step6` |

**行为**

- `onboarding.skip` / `onboarding.later`：结束教程，本地标记 `tutorial_completed` 或 `tutorial_dismissed`。  
- 设置中 `onboarding.replay`：重置标记并重新播放。  
- 埋点事件名建议：`tutorial_step_view`, `tutorial_step_next`, `tutorial_skip`, `tutorial_replay`（与 PRD 埋点章节对齐）。

---

## 5. 关键组件样式要点

- **按钮**：主按钮实心 accent；次按钮白底 + border。  
- **输入框**：白底、细 border，focus ring 用 accent 浅色。  
- **问题列表项**：左侧小状态点（未开始/进行中/已完成），选中行 `surface-muted`。  
- **对话气泡**：用户右对齐浅底；助手左对齐白底 + 细边框。
